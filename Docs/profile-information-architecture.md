# Profile Information Architecture

## Goal
`Profile` 탭을 개인 관리 허브로 사용하고, `Settings` 화면은 세부 설정 허브로 역할을 분리한다.

## Profile Root
- 루틴: 루틴 관리 화면으로 진입
- 빠른 설정: `하루 목표량`, `메인 아이콘`을 바로 수정
- 계정: `회원 탈퇴` 진입
- 앱 정보: 버전과 빌드 번호 확인
- 현재 설정: 자주 확인하는 개인화 값 요약

## Settings Screen
- 개인화: `하루 목표량`, `메인 아이콘`
- 계정 관리: `회원 탈퇴`

## Role Separation
- `Profile`은 자주 접근하는 개인 관리 액션과 정보 요약을 제공한다.
- `Settings`는 전체 설정을 모아두는 세부 설정 허브 역할을 유지한다.
