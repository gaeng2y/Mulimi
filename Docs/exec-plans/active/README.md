# Active Exec Plans

진행 중인 작업 계획을 두는 폴더다.

## Naming

- 권장 형식: `YYYY-MM-DD-issue-###-short-title.md`
- 예시: `2026-04-07-issue-165-permission-copy-localization.md`

## Minimum Template

```md
# Title

## Context

## Goal

## Constraints

## Plan
1.
2.
3.

## Validation

## Open Questions
```

## Rules

- 이슈 본문을 그대로 복사하지 말고 결정과 작업 순서를 적는다.
- 구조 변경이면 관련 문서 업데이트 항목을 명시한다.
- 완료되면 `completed/`로 이동한다.
